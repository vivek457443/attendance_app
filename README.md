# Attendance Management System

## 📌 Project Overview

The **Attendance Management System** is a web-based application developed to manage student attendance efficiently.
It allows an **admin to manage student records and mark attendance**, while students can **view their attendance status and profile**.

This project demonstrates the use of **Python backend development with Flask**, database management, and frontend integration using HTML, CSS, and Bootstrap.

---

## 🚀 Features

### 👨‍💼 Admin Panel

* Admin login authentication
* Add and manage students
* Mark **Entry Time (In-Time)** for students
* Mark **Exit Time (Out-Time)** for students
* View attendance records

### 👨‍🎓 Student Dashboard

* View personal profile
* Check attendance status
* See **entry time and exit time**
* View attendance history

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/YOUR_USERNAME/attendance_app.git
```

### 2️⃣ Navigate to project folder

```
cd attendance_app
```

### 3️⃣ Install dependencies

```
pip install flask
```

### 4️⃣ Run the application

```
python app.py
```

### 5️⃣ Open in browser

```
http://127.0.0.1:5000
```
